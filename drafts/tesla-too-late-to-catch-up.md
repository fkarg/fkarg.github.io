So I read [this][article] article some time ago.


Basic Idea: When Tesla starts testing new features, people complain that there
is not enough regulation regarding it, and that it is dangerous and whatnot.

With this at least being somewhat true, it is likely that forms of regulation
will happen soon. Regulations, that will not allow others to do the same. That
will prevent others from collecting the data that is necessary to at some point
solve FSD. prevent in the sense that it does not allow for a similarly fast
'trial and error'-prone approach, in favour of a severely slower and
'confident' one. While it would be safer, it would take an increadible amount
of more time to completion. maybe enough for them to go bankrupt, or, more
likely, long enough so that TESLA already has total dominance.
