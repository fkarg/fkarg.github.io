# The Goal
The goal is to learn as much as possible about docker in the shortest
time possible, and have fun while doing it.


## Docker
So my guess is that you have at least some knowledge about docker when reading
it, at least heard the name at some point, or someone mentioned it when talking
about it.

Obviously, first is installing docker, either with `snap` or with `apt`, if you
are on ubunt-based systems.

I did a `snap install docker`, and got version `18.09.9`. Next, you might want
to run `docker run hello-world` to verify your setup.
